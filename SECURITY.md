# Security Policy

## Scope

AletheIA is a framework for governed AI-assisted work.

Security is therefore part of the framework itself, not an afterthought.

## Please report

Please report issues involving:

- unsafe execution paths
- policy bypasses
- traceability gaps
- unsafe defaults
- accidental exposure of secrets or sensitive context

## Responsible disclosure

Please do not open public issues for sensitive vulnerabilities.

Use a private channel or repository contact method when available.
