# Task Checklist

Use this checklist before concluding any relevant AI-assisted task.

## Before starting

- Is the task small enough and clearly scoped?
- Is there a clear objective?
- Are boundaries explicit?
- Is the dominant front clear?

## Before execution

- Was the system analyzed before changing anything?
- Is there an explicit plan?
- Are allowed files and forbidden files clear?
- Are contracts or source-of-truth artifacts identified?

## Before closing

- Was the change validated?
- Were core regressions checked?
- Does the result still match the source of truth?
- Is a handoff needed?
- Did this task produce a reusable learning?
