This document exists to help present AletheIA clearly in public.

## 1. GitHub short description

**Operating framework for AI-assisted work with decision, governance, validation, and learnings before execution.**

## 2. Short tagline

**From output to governed action.**

## 3. One-paragraph project description

AletheIA is an operating framework for AI-assisted work. It helps teams structure context, decisions, governance, validation, and learnings before execution, so model output does not act directly on systems without control. The framework is provider-agnostic, deterministic at its core, and designed to be reusable across projects.

## 6. Announcement copy — longer

### English

AletheIA is our attempt to move beyond the common `prompt -> output -> execution` pattern.

Instead of letting raw model output act directly on the system, AletheIA introduces an operating layer for:

- context discipline
- structured decision-making
- governance and policy checks
- proportional validation
- reusable learnings

The repository is now public as an early alpha draft. The goal of this first version is clarity over completeness and structure over cleverness.

### Português

O AletheIA é nossa tentativa de ir além do padrão comum de `prompt -> output -> execution`.

Em vez de permitir que o output bruto do modelo aja diretamente no sistema, o AletheIA introduz uma camada operacional para:

- disciplina de contexto
- decisão estruturada
- governança e policy checks
- validação proporcional
- learnings reutilizáveis

O repositório agora está público como um alpha inicial. O objetivo desta primeira versão é priorizar clareza sobre completude e estrutura sobre esperteza.
