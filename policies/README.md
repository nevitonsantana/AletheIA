# AletheIA — Policy Packs

## Objetivo

Esta pasta reúne policy packs do `AletheIA`.

Um policy pack é um conjunto versionável de regras declarativas que pode ser interpretado por uma camada de governança do framework.

---

## O que entra aqui

- packs JSON de políticas
- versões por domínio
- regras específicas de governança

---

## Regra importante

Policy packs não são o framework inteiro.

Eles são uma camada de disciplina operacional que o `AletheIA` pode usar para:

- bloquear
- pedir revisão
- exigir human gate
- permitir continuidade
